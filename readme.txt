== Description ==

This plugin allows you to easily integrate https://aads.com/ banner advertisement into your website using WordPress Blocks or legacy widgets.
It's as simple as drag and drop. 


== Frequently Asked Questions ==

= Does it support adaptive ad units =

This plugin is capable of using all sizes of AADS ad units, including "Adaptive".
